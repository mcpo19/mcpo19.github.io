# Multiples of 3 or 5 [solved]

# Problem statement

If we list all the natural numbers below 10 that are multiples of 3 or 5, we get 3, 5, 6 and 9. The sum of these multiples is 23. Find the sum of all the multiples of 3 or 5 below 1000.
Once completed, create a function to automate the process — i.e. receives any integer to represent the limit and sums all the multiples of 3 or 5 until limit.

This challenge serves today and Friday. Solution(s) will be shared on Monday (October 7th, 2019)

# Solution

The trick in this challenge was to use the modulus operator %. It gives us the remainder of a calculation — e.g. 10 % 2 results in 0 and 10 % 3 results in 1. This way, all we had to do was loop through all the numbers (until the limit) checking if the number had a remainder 0 with either 3 or 5 — e.g. i % 3 == 0 or i % 5 == 0. If true, the number should be added to the total sum.

So, a possible solution was:

def multiply_3s_and_5s(limit):
    total = 0
    for i in range(limit):
        if i % 3 == 0 or i % 5 == 0:
            total += i
    return total


# Checking result:
result = multiply_3s_and_5s(1000)
print(result)
